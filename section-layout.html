4 Layout Layout

Decanter comes with not one but two grid systems.

Decanter has complimentary flex box and css grid based grid systems. As browsers mature and the user base updates, we may see a winner for which one is more widely used, or, more likely, we will see standardization around the use cases for each. CSS Grid and Flex Box have different strengths and should be used where it makes sense. Here are a few articles outlining the differences and strengths of each.

Resources: https://hackernoon.com/the-ultimate-css-battle-grid-vs-flexbox-d40da0449faf https://codeburst.io/css-grid-vs-flexbox-building-simple-layouts-43e569a66f0a https://medium.com/youstart-labs/beginners-guide-to-choose-between-css-grid-and-flexbox-783005dd2412
